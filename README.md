# Prime Tech Engineering Website

A modern, responsive corporate website for **Prime Tech Engineering**, designed to showcase engineering solutions, services, and company information with a clean and engaging user interface.

## 🌐 Overview

This project is a responsive corporate website developed to present the company's engineering services, expertise, and contact information in a professional and user-friendly manner.

## ✨ Features

- Responsive design for desktop, tablet, and mobile
- Modern corporate landing page
- Smooth scrolling navigation
- Interactive user interface
- Professional services section
- About company section
- Contact section
- Custom branding and favicon support
- Optimized assets and clean layout

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/primetech-website.git
```

### Open the Project

Navigate to the project folder:

```bash
cd primetech-website
```

Open `index.html` in your preferred web browser, or use a local development server.

### Using VS Code Live Server

1. Install the **Live Server** extension.
2. Right-click on `index.html`.
3. Select **Open with Live Server**.
